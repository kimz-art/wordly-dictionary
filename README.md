# Wordly Dictionary App
## Description
A simple browser-based dictionary app that lets you search for any English word and see its full definition, pronunciation, audio, synonyms, antonyms, and usage examples 
## Features
Definitions — all definitions listed per part of speech
Phonetic pronunciation — displayed next to the word
Audio pronunciation — play button when available
Examples — shown beneath each definition where available
Synonyms & Antonyms — listed per meaning
## How to run the project
1. Clone the repository
2. Open the project folder 
3. Open the index.html file in your browser 
## Technologies Used 
- HTML
- Javascript
- CSS
- Fetch API(JSONplaceholder API)
